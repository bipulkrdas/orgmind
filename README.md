# OrgMind - Enterprise Document Processing Platform

OrgMind is an enterprise-grade document processing platform that transforms documents into intelligent knowledge graphs. Users can upload documents, create knowledge graphs, and interact with their data through AI-powered chat powered by Google Gemini and Zep Cloud.
